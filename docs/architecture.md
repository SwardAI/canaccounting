# Architecture

## System Overview

A simple, modern web application with three core components: a marketing site, Stripe payment integration, and an AI chatbot. Designed for fast development and easy maintenance.

```
┌─────────────────────────────────────────────┐
│                   Client                     │
│              (Browser / Mobile)              │
└─────────────┬───────────────────┬───────────┘
              │                   │
              ▼                   ▼
┌─────────────────────┐ ┌─────────────────────┐
│   Next.js Frontend  │ │    AI Chatbot UI     │
│   (Landing Page,    │ │   (Embedded widget   │
│    Pricing, FAQ)    │ │    on the site)      │
└──────────┬──────────┘ └──────────┬──────────┘
           │                       │
           ▼                       ▼
┌─────────────────────┐ ┌─────────────────────┐
│   Stripe Checkout   │ │  Anthropic/OpenAI    │
│   (Payment)         │ │  API (Chat backend)  │
└──────────┬──────────┘ └─────────────────────┘
           │
           ▼
┌─────────────────────┐
│  Email Service       │
│  (Post-purchase      │
│   confirmation)      │
└─────────────────────┘
```

## Tech Stack

| Layer        | Technology                  | Why                                              |
|-------------|----------------------------|--------------------------------------------------|
| Framework   | Next.js 15 (App Router)     | Fast, SEO-friendly, easy deployment on Vercel     |
| Language    | TypeScript                  | Type safety, better developer experience          |
| UI Library  | shadcn/ui + Radix UI        | Beautiful, accessible, customizable components    |
| Styling     | Tailwind CSS v4             | Utility-first, pairs perfectly with shadcn/ui     |
| Animations  | Framer Motion               | Production-grade scroll reveals, hover effects, transitions |
| Icons       | Lucide React                | Clean icon set, recommended by shadcn/ui          |
| Payments    | Stripe Checkout             | Secure, trusted, minimal backend code needed      |
| Chatbot     | Anthropic Claude API        | High quality responses, easy to set up            |
| Hosting     | Vercel                      | Free tier, instant deploys, great with Next.js    |
| Email       | Resend                      | Simple API, good free tier, works well with Next.js|

## Key Flows

### Payment Flow
1. User selects a pricing tier → clicks "Buy Now"
2. Frontend calls `POST /api/checkout` with the selected tier
3. Backend creates a Stripe Checkout Session with the correct price
4. User is redirected to Stripe's hosted checkout page
5. After payment, Stripe redirects to `/success` page
6. Stripe webhook (`/api/webhook`) fires → triggers confirmation email via Resend
7. Email includes next steps (what documents to send, contact email)

### Chatbot Flow
1. User clicks chat widget → chat window opens
2. User types a question
3. Frontend sends message to `POST /api/chat`
4. Backend forwards to Anthropic API with a system prompt containing:
   - Service details, pricing info, FAQ answers
   - Instructions to be helpful but direct users to purchase
   - Instruction to say "please email us at X" for complex/specific questions
5. AI response streamed back to the chat UI

### Post-Purchase Flow
1. Client receives confirmation email with instructions
2. Client emails required documents to the team
3. Can handles the accounting work offline
4. All further communication via email (no website portal needed)

## Deployment

- Push to GitHub → auto-deploy on Vercel
- Environment variables set in Vercel dashboard
- Custom domain configured in Vercel
- Stripe webhook endpoint pointed to production URL

## Security Considerations
- All API keys server-side only (never exposed to client)
- Stripe handles all payment data (PCI compliant)
- AI chat API route proxied through backend (API key not exposed)
- Stripe webhook signature verification on `/api/webhook`
- No user data stored in a database (MVP — all handled via email)
- HTTPS enforced via Vercel
